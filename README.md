# webpack-laravel-mix

Clone and enter to project

```sh
git clone https://github.com/xander-crutch/webpack-laravel-mix.git
cd webpack-laravel-mix
```

Install the dependencies and devDependencies.

```sh
npm install
```

Run development

```sh
npx mix
```

or

```sh
npm run development
```

Run dev and watch, start server

```sh
npx mix watch
```

or

```sh
npm run watch
```

Run production

```sh
npm run production
```
